## Structure

1. "./processVideo": BPM extracting script.
2. "./result": Extracted BPM data.
3. "./server": Server to receive videos for BPM extraction.
4. "./videos": Videos for BPM extraction.
5. "./wpd_on_phone": App to record and send video.
